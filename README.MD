# CREOSON的测试代码

## 1. CREOSON的环境配置

**首先确保JLINK已正确安装**。在官网下载 `CREOSON Server`,地址在<a href="https://github.com/SimplifiedLogic/creoson/releases" target="_blank">https://github.com/SimplifiedLogic/creoson/releases</a>。下载解压后，运行目录下`CreosonSetup.exe`,在 `Step 1:`栏输入正确的CREO安装目录，点击 `Start CREOSON`，就可以启动CREOSON服务，

## 2.test.py

- Python测试代码，打开当期目录下fin.prt添加参数后保存。

## 3.test.js

- Nodejs测试代码，打开当期目录下fin.prt添加参数后保存。
- creoson_js的文件复制自`CREOSON Server``\web\assets\creoson_stuff\creoson_js`，加入模块导出。
